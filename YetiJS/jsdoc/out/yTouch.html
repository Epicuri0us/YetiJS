Class: yTouch

yTouch

The touch events save the state of a touch event. It is used by the input manager.

new yTouch

Properties:
Name Type Description
isTouched Boolean Tells, if the touch is active.
wasTouched Boolean Tells, if the touch was active last frame.
position yVector The position of the touch.
lastPosition yVector The position of the touch last frame.
wasMoved Boolean Tells, if the position of the touch changed from last frame.
