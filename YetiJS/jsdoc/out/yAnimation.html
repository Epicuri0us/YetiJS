Class: yAnimation

yAnimation

The animation-object saves all the information and provides all methods to play sprite animations

new yAnimation

Parameters:
Name Type Description
pSpriteSheet ySpriteSheet The spritesheet.
pFrameTime Integer The length in milliseconds a frame should be displayed.
pFirstFrame Integer The ID on the spritesheet where the animation should begin.
pLastFrame Integer The ID on the spritesheet where the animation should end.
pLoop Boolean (optional) Tells, if the animation should loop (Default: false).
Properties:
Name Type Description
spriteSheet ySpriteSheet The spritesheet.
frameTime Integer The length in milliseconds a frame is displayed.
firstFrame Integer The ID on the spritesheet where the animation begins.
lastFrame Integer The ID on the spritesheet where the animation ends.
activeFrame Integer The ID of the active frame.
loop Boolean Tells, if the animation loops.
stopped Boolean Tells, if the animation is running.
lastFrameTime Integer Used to determine if the next frame should be drawn.
