Class: yTimer

yTimer

The timer class is used for countdown purposes and is able to execute a function when time is elapsed.

new yTimer

Parameters:
Name Type Description
pDuration Integer The duration of the timer in milliseconds.
pDuration Integer The duration of the timer in milliseconds.
Properties:
Name Type Description
duration Integer The duration.
startTime Integer The time when the timer started.
remainingTime Integer The remaining time.
paused Boolean Tells, if the timer paused, Default is true;
elapsed Boolean Tells, if the timer elapsed.
callback Function (optional) The function that should be executed when the timer elapses.
