yObjectPool

The yObjectPool is a generic object pool used to minimize garbage creation on runtime.

new yObjectPool

Parameters:
Name Type Description
pObjectClass Class The class of the objects in the pool.
pSize Integer The initial size of the pool.

Properties:
Name Type Description
objectClass Class The class of the objects in the pool.
freeObjects Object[] The array containing all free objects at the time.
