Class: yLayer

yLayer

A layer handles the game functionality and should be dedicated to one task (i.E. user inferface/menu/actual game/...), it has to be inherited and attached to a specific gamestate.

new yLayer

Parameters:
Name Type Description
pInactive Boolean (optional) Tells that the layer is inactive.
Properties:
Name Type Description
active Boolean Checks if the layer is active.
camera yCamera The active camera.
physWorld b2World The physics world.
physDelta Integer Delta to calculate when to simulate the next physics step.
entities yEntity[] The active camera.
