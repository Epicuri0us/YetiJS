Class: yPhysicsObject

yPhysicsObject

The physics object saves physical data and collision models for very basic physics calculation. Use Box2DWeb for more accurate and advanced physics.

new yPhysicsObject

Parameters:
Name Type Description
pPosition yVector The position.
pWeight Float The weight.
Properties:
Name Type Description
position yVector The position.
velocity yVector The active velocity.
friction Float The friction.
weight Float The weight.
cor Float The "coefficient of restitution".
model yPolygon The polygon that represents the object.
collisions yPhysicsObject An array that stores every physics object this object collides with, clears every frame.
